# AndroidTinderSwipe
The project demonstrates the swipe similar to tinder app.

[![IMAGE ALT TEXT HERE](http://img.youtube.com/vi/xUQSAYGzj1Q/0.jpg)](http://www.youtube.com/watch?v=xUQSAYGzj1Q)

and add views again.
